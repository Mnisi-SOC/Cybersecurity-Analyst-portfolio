# Cybersecurity-Analyst-portfolio

## About Me
I'm a Computer Science student passionate about cybersecurity, digital forensics, and SOC operations. My focus is on learning how to detect, analyze, and respond to cyber threats using hands-on tools and simulations.

This repository documents my practical progress and learning journey toward becoming a SOC Analyst. It includes reports, analyses, and lab documentation from a structured 6-week roadmap designed to build cybersecurity skills.

- **Networking:** TCP/IP, OSI Model, DNS, HTTP, Wireshark, TCPDump
- **Security Tools:** Splunk, Wazuh, Nmap, VirusTotal, Any.Run
- **Scripting:** Python (for automation and log parsing)
- **Key Competencies:** Log analysis, Threat detection, Incident response, Phishing analysis
